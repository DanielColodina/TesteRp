# Módulo: Estoque / Almoxarifado

## Funcionalidades Principais
- **Cadastro de Materiais**: Código, descrição, unidade, fornecedor, preço médio.
- **Entrada de Materiais**: Registro de compras, com NF-e, quantidade, valor.
- **Saída de Materiais**: Alocação para obras, com justificativa e responsável.
- **Estoque Mínimo**: Configuração de alertas quando estoque baixa.
- **Relatórios de Consumo por Obra**: Quantidade usada por obra, com datas.

## Funcionalidades Avançadas
- **Inventário Periódico**: Contagem física com discrepâncias.
- **Categorização**: Materiais por tipo (cimento, ferro, etc.).
- **Integração com Compras**: Sugestões automáticas de reposição.

## Interface
- Lista de materiais com filtros por obra.
- Alertas visuais para itens críticos.
- Formulários simples para entradas/saídas.

## Fluxos de Uso Diário
- Almoxarife registra entrada de materiais ao receber entrega.
- Registra saída ao liberar para obra, com assinatura digital.
- Verifica alertas de estoque mínimo e solicita reposição.
